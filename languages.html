Ruby

1995. Yukihiro "Matz" Matsumoto

  • Uses: Application, scripting, web, infrastructure as code
  • Parent language/written in: C
  • Common Applications: Chef, Metasploit Framework, Ruby on Rails
  • Notes: Gaining popularity with Infrastructure as Code
  • # of Indeed jobs listed: 19,676
  • Hello World sample:
puts "Hello World"


Java

May 23,1995. James Gosling at Sun Microsystems

  • Uses: Application, business, client-side, general, mobile development, server-side, web
  • Parent language/written in: Sun JVM => C; Oracle JVM => C++ ; Java Compiler by Oracle => Java
  • Common Applications: Android applications, OpenOffice, NetBeans, Eclipse, Vuze
  • Notes: Java is a very popular cross-platform language. If you are serious about programming, you'll learn Java, C, or C++(hopefully a little of each). Java is less susceptible to buffer overflows than C and C++.
  • # of Indeed jobs listed: 67,681
  • Hello World sample:
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}


Python

December 1989. Released to public Feb 20, 1991. Guido van Rossum at Centrum Wiskunde & Informatica (CWI)

  • Uses: Application, general, web, scripting, artificial intelligence, scientific computing
  • Parent language/written in: CPython, the first implementation was written in C.
  • Common Applications: Youtube, Dropbox, Google, Quora, Instagram, BitTorrent, Spotify, Reddit
  • Notes: Python is a powerful general purpose, interpretted language. it's know for flexibility and speed. commonly taught in computer science programs for it's ease of use.
  • # of Indeed jobs listed: 45,136
  • Hello World sample:
print("Hello World!")


.NET

Feb 13, 2002. Microsoft

  • Uses: F#:applications ; C#:application, business, client-side, general, server-side, web ; Visual Basic: Applications, education, web, business, general
  • Parent language/written in: Originally C & C++
  • Common Applications: Visual Studio, KeePass, Low Orbit Ion Cannon(DDoS), Paint.NET, Windows installer
  • Notes: .NET is a development platform, not a language. VB script & C# are microsoft languages created for the platform. also includes Managed C++, F#. ASP is the part of the .NET library for scripting web pages. Using the .NET platform, allows these languages to talk to each other.
  • # of Indeed jobs listed: ".NET" 56,877
  • Hello World sample:
using System;
using System.Collections.Generic;
using System.Text;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, world!");
            Console.ReadLine();
        }
    }
}


PHP

1994. Rasmus Lerdorf

  • Uses: Server-side, web application, web
  • Parent language/written in: C
  • Common Applications: parts of Facebook, Wikipedia, Yahoo!, Wordpress, Drupal, early Youtube
  • Notes: Personal Home Page ==> PHP: Hypertext Preprocessor
  • # of Indeed jobs listed: 16,604
  • Hello World sample:
<html>
  <head>
    <title>PHP Test</title>
  </head>
  <body>
    <?php echo '<p>Hello World</p>'; ?> 
  </body>
</html>
